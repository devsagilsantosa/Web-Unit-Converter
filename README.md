# Web-Unit-Converter
Unit Converter – JavaScript Practice

This project is a practice exercise for learning JavaScript, HTML, and CSS.
It is a simple unit converter application that converts numeric input into different measurement units.

Description

The application allows users to enter a number and convert it into several units, including:

Length: Meter ⇄ Feet

Volume: Liter ⇄ Gallon

Mass: Kilogram ⇄ Pound

The main purpose of this project is to practice:

DOM manipulation using JavaScript

Using addEventListener

User input validation

Separating logic into reusable functions

Using constant objects for conversion factors

Formatting output for better readability

Features

Number input validation (cannot be empty, non-numeric, or negative)

Two-way unit conversion for each category

Conversion results displayed with 3 decimal places

Input field is automatically cleared and focused after conversion

Technologies Used

HTML

CSS

JavaScript (Vanilla JS)

Notes

This project is created for learning purposes only and is not intended for production use.
The focus is on understanding JavaScript fundamentals and writing cleaner, more reusable, and maintainable code.
